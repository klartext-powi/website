INHALT
  favicon.ico ............. klassisches Favicon (16/32/48 px), helle Variante
  favicon-dark.ico ........ dunkle Variante
  icon/ ................... PNGs in 16/32/48/180/512 px (hell + dunkel)
  wordmark/ ............... Wortmarke als transparente PNGs (hell + dunkel)
  svg/ .................... alle Vektor-Originale (verlustfrei skalierbar)

VERWENDUNG
  hell  = fuer helle Hintergruende   dunkel = fuer dunkle Hintergruende
  16/32 px ... Browser-Tab    180 px ... Apple Touch / Homescreen    512 px ... PWA / App
